Capability Map

图片内容的逻辑关系拆解

01

全栈架构是底座

微服务、缓存、数据库、认证和网关先保证系统跑得稳，才能支撑海外 C 端高并发和多入口接入。

02

Agent 是核心产品价值

规则引擎与大模型决定“会不会聊”，Milvus 记忆与风险判断决定“能不能持续聊、放心聊”。

03

支付与会员是商业变现

免费对话额度负责拉新试用，Stripe 订阅负责将高频用户转为会员收入，形成 Web 支付闭环。

04

合规是全球运营前提

数据脱敏、删除导出和地区策略不是附加功能，而是出海产品在 GDPR 和北美隐私法下的准入门槛。
