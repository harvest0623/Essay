# Global AI Agent Platform Demo

这是一个零依赖的前端演示 demo，用来把需求图片中的四部分能力串成完整业务闭环：

- 全栈架构与后端核心体系
- AI Agent 核心逻辑与能力落地
- 海外业务闭环与合规体系
- 跨团队协同与技术统筹

## Demo 内容

页面聚焦三个问题：

1. 这套系统由哪些模块组成
2. 这些模块之间如何串联成产品链路
3. 为什么它能够支撑“海外社交流量 + AI Agent + 付费转化 + 合规运营”

主要展示区块：

- 架构分层图
- 用户旅程 / 业务闭环步骤
- Agent 交互场景模拟
- 能力逻辑拆解

## 如何运行

这是纯静态页面，无需安装依赖。

直接用浏览器打开 `index.html` 即可。

如果你希望通过本地服务启动，也可以在当前目录执行：

```powershell
npx serve .
```

## 逻辑说明

这个 demo 将图片里的内容整理成如下主链路：

社交渠道接入
→ 统一身份认证
→ API Gateway 标准化请求
→ Agent 编排与大模型调用
→ Milvus 长期记忆与语义检索
→ 风险识别与关系状态判断
→ 免费额度与 Stripe 会员转化
→ GDPR / 隐私合规治理

## 文件结构

- `index.html` 页面结构
- `styles.css` 页面样式
- `script.js` 交互逻辑与演示数据
